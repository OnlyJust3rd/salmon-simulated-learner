SQL is the standard language for querying a relational database. A SELECT
statement retrieves rows and columns from one or more tables. The FROM clause
names the table and the WHERE clause filters rows by a predicate. Projection
chooses which columns appear in the result while selection restricts which rows
are returned. The query produces a result set built from the underlying table
data.
